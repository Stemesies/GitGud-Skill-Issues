# GitGud: Skill Issues
Реализация аналога Github Issues с помощью Angular, стремящаяся воссоздать в полной мере этот незабываемый опыт работы с задачами в Github ~~

## Текущие возможности
На данный момент GitGud: Skill Issues способен:
* Показывать список задач
* Позволять пользователю гибко сортировать, искать и фильтровать задачи
* Создавать новые задачи
* Регистрировать и авторизовывать пользователя с помощью простой системы авторизации
* Вести историю всех изменений задачи от лица текущего аккаунта, включая:
  * Комменты
  * Переименования
  * Открытие, закрытие задач
  * Смена приоритета и исполнителей задачи
  * Смена тегов задач
* Оповещать пользователя о успешных (или нет) операциях с помощью всплывающих уведомлений
* Как в самом Github показывать помимо абсолютного времени событий относительное, что поможет крайне легко определить что когда было сделано. 

И все это приправлено визуалом, крайне схожим с оригиналом. 

## Запуск
Склонируйте репозиторий и запустите локальный сервер с помощью
```bash
ng serve
```

## Детали реализации
На данный момент проект использует 5 сервисов:
* AccountLogger - управление созданными аккаунтами и входом
* IssueManager - управляет списком всех задач
* IssueLogger - ведение истории задачи
* Label service - управляет списком всех тегов
* Toast service - Управляет всплывающими уведомлениями. 

Проект не использует Angular Material (зато использует кое-что гораздо круче)

RxJS используется для реактивного получения информации о текущем аккаунте и для логистики отображения уведомлений.
