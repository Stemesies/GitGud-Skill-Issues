# GitGud: Skill Issues
Реализация раздела Github Issues с помощью Angular, стремящаяся в полной мере воссоздать этот незабываемый опыт работы с задачами в Github ~~

## Текущие возможности
На данный момент GitGud: Skill Issues способен:
* Показывать список задач
* Позволять пользователю гибко сортировать, искать и фильтровать задачи
* Создавать новые задачи и теги
* Регистрировать и авторизовывать пользователя с помощью простой системы регистрации и авторизации
* Вести историю всех изменений задачи от лица текущего аккаунта, включая:
  * Комментарии
  * Переименования задач
  * Открытие, закрытие задач
  * Смена приоритета и исполнителей задачи
  * Смена тегов задач
* Оповещать пользователя об успешных (или нет) операциях с помощью всплывающих уведомлений
* Как и в самом Github показывать помимо абсолютного времени событий относительное, что поможет крайне легко определить что когда было сделано. 

И все это приправлено визуалом, крайне схожим с оригиналом. 

## Запуск
Склонируйте репозиторий и запустите локальный сервер с помощью
```bash
ng serve
```

## Детали реализации
На данный момент проект использует 5 сервисов:
* AccountLogger - управление созданными аккаунтами и входом
* IssueManager - управляет списком всех задач
* IssueLogger - ведение истории задачи
* Label service - управляет списком всех тегов
* Toast service - Управляет всплывающими уведомлениями. 

Проект не использует Angular Material (Его сюда тупо не впихнуть, извините)

RxJS используется для реактивного получения информации о текущем аккаунте, для логистики отображения уведомлений и т.д.
